# Macwhisper Meet Reminder
Google Chrome extension which sends a notification reminder whenever a Google Meet tab gets opened, to record it with MacWhisper.

![macwhisper-meet-reminder](https://github.com/user-attachments/assets/133b4682-14f3-4624-89cf-46cd8c0ecce5)

## Instructions:
1. Go to chrome://extensions/
2. Make sure developer mode is enabled
3. Upload the extension folder/files (Load unpacked)
4. Activate
